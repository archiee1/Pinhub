# 📌 PinHub

**PinHub** is a Pinterest-inspired image-sharing web application where users can upload, view, and save aesthetic image posts. Built using the MERN stack (without React) and styled with Bootstrap & custom CSS, it delivers a smooth and responsive UI with a functional backend.

---

## 🚀 Features

- 🖼️ Upload images with captions
- 📋 View all image posts in a Pinterest-style grid
- ❤️ Like or save posts (optional feature)
- 📂 Organized and responsive layout using Bootstrap
- 🌐 Fully dynamic backend with MongoDB + Express + Node.js
